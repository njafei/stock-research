## 一、行业结构分析（权重20%）评分：7.0/10

### 行业定位
- **细分行业**：调味发酵品-榨菜/佐餐开胃菜
- **行业规模**：国内榨菜市场约200-300亿元，规模较小但稳定
- **竞争格局**：涪陵榨菜为行业绝对龙头，市场份额约40-50%，"乌江"品牌认知度极高

### 行业特征
| 指标 | 评价 |
|------|------|
| 市场规模增速 | 低个位数增长，行业进入成熟期 |
| 竞争激烈程度 | 低集中度，但乌江品牌壁垒显著 |
| 周期性 | 弱周期，必选消费抗经济波动 |
| 进入壁垒 | 品牌+渠道+青菜头产地优势三重壁垒 |

### 核心判断
榨菜是典型的"小产品、大公司"赛道。行业增速有限但竞争格局稳定，乌江牌市占率遥遥领先。短期受消费疲软影响需求平淡，中长期看产品升级与渠道下沉仍有空间。

## 二、护城河评估（权重25%）评分：7.5/10

### 护城河类型：品牌护城河 + 地理护城河

**1. 品牌护城河（强）**
- "乌江"牌榨菜为品类代名词，消费者心智占领深厚
- 多次被评为中国榨菜标志性品牌，品牌溢价能力强
- 定价权：榨菜单价约1-2元，价格敏感度低，50%+毛利率验证品牌力

**2. 地理护城河（中等）**
- 青菜头产自重庆涪陵地区，独特的地理气候形成品质壁垒
- 原料产地优势降低运输成本，形成天然保护圈

**3. 渠道护城河（较强）**
- 覆盖全国2640+经销商，下沉到县级市场
- 商超、农贸、电商多渠道覆盖
- 2024年主动优化经销商结构，削减607家布局不合理的经销商

**4. 规模护城河（中等）**
- 年销售榨菜约11万吨，规模效应显著
- 自动化产线降低单位成本

## 投资建议

### 亮点
1. **绝对龙头**：乌江牌榨菜市场占有率第一，品牌护城河深厚
2. **财务极健康**：现金68亿，零有息负债，分红率60%
3. **估值合理**：动态PE约17倍，处于历史中位偏低
4. **2025年恢复预期**：机构预期净利润9.05亿，同比+13%

### 顾虑
1. **成长性弱**：连续2年营收负增长，品类单一问题突出
2. **ROE下降趋势**：从12%降至9.44%，需观察能否企稳
3. **行业天花板**：榨菜市场容量有限，需要新增长极

### 结论
**评级：值得关注（6.8/10）**

涪陵榨菜是一家质地优秀的细分龙头公司——品牌强、现金好、分红高，但成长性是最大软肋。动态PE17倍、ROE9.44%，适合作为稳健现金牛配置而非高增长标的。2025年收入目标+8%能否实现是关注焦点，若新品（萝卜、泡菜、榨菜酱）放量超预期，可能带来估值重塑机会。
